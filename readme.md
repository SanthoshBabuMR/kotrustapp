# iTrust

Web based application

### Version
1.0.0

### Dependencies

You need to have NodeJS installed:
* [download nodeJS]

### Download application

```sh
$ git clone https://github.com/html5webster/kotrustapp.git kotrustapp
```

### Launch Server
```sh
$ cd kotrustapp
$ node index
```

 [download nodeJS]: <https://nodejs.org/en/>
